LibApps API as of 14 August 2017

For Reference Purposes

Source:https://lgapi-us.libapps.com/1.2/az

Notes

URL Parameters

Name: id optional Type: Numeric Values: Asset id(s) Description: If an ID is present the asset with that ID will be retrieved. Multiple IDs can be specified with a comma delimited list. Ex: az/1,2

Example

Authentication Request

POST /1.2/oauth/token HTTP/1.1 Host: lgapi-us.libapps.com client_id=250&client_secret=bca31c3811d383d7b1c582e3c435fc77&grant_type=client_credentials

Response

{ "access_token":"f65f53c20096e3dba63007e7dc8cbb539cf6e4bf", "expires_in":3600, "token_type":"Bearer", "scope":"az_get az_create az_update" }
